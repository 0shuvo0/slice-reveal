Slice-Reveal

Get Started Usage Demo

1. Download the library

scroll-reveal.zip

2. Including the CSS file

<link rel="stylesheet" href="css/slice-reveal.min.css">

3. Including the JavaScript file

<script src="js/slice-reveal.min.js"></script>

4. Initialization

initSliceReveal();
/*or with options(Intersection Observer options)*/
const ops = {
    rootMargin: "0px 0px -50px 0px"
}
initSliceReveal(ops);

1. Add slice-reveal class to desired elements

<div class="slice-reveal">Hello world</div>

2. You can use any combination of available classes

reverse, horizontal, reverse-order, alternate

3. Optionally you can add some attributes to modify behaviour

<div class="slice-reveal reverse horizontal" num-slice="5" close-revealed="true">Hello world</div>

4. Available attributes:

num-slice: Define number of slices

reveal-delay: Delay between each slice reveal

reveal-duration: Define slice reveal animation duration

close-revealed: Close slices once parent gets out of view

slice-color: Define color of slices

*Using the slim build

<script src="js/slice-reveal.slim.min.js"></script>

*What is slim build

If your using Intersection Observer or some other scroll reveal library then you don't need the extra codes that slice-reveal ships with in that case you can the slim build it does not incdule intersection observer functionality. So you have to add active class by yourself. And you'd still have to call initSliceReveal() function.
